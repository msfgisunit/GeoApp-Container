1. Download ZIP File on your computer
2. Unzip archive
3. Places the GeoApp web files in the MSF-GeoApp-Container\resources\app\src\container folder
    The index.html file must be at the root of this folder
4. Launch your GeoApp by opening the MSF-GeoApp-Container\MSF-GeoApp-Container.exe file
5. If you want to launch another GeoApp, just replace the files in the MSF-GeoApp-Container\resources\app\src\container folder
